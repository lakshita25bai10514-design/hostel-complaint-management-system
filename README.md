# Hostel Complaint Management System (GUI Version)

This project is a GUI-based application built using Python and Tkinter to manage hostel complaints efficiently.

## Features
- User Login System (Admin & Student)
- Add Complaint
- View Complaints
- Resolve Complaints (Admin only)

## Technologies Used
- Python
- Tkinter (GUI)
- JSON (Data Storage)

## How to Run
1. Install Python
2. Run:
   python main.py
