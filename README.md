# 🏨 Hostel Complaint Management System (GUI Version)

The **Hostel Complaint Management System** is a GUI-based application developed using **Python** and **Tkinter**. It provides an efficient and structured way for students to register complaints and for administrators to manage and resolve them.

This project replaces traditional manual complaint systems with a digital solution, improving transparency, organization, and communication.

---

## 🚀 Features

* 🔐 User Login System (Admin & Student)
* 📝 Add Complaint
* 📋 View Complaints
* ✅ Resolve Complaints (Admin Only)
* 💾 Data Storage using JSON
* 🖥️ Simple and User-Friendly Interface

---

## 🛠️ Technologies Used

* **Python** – Core programming language
* **Tkinter** – GUI development
* **JSON** – Data storage and retrieval

---

## ▶️ How to Run the Project

1. Install Python on your system
2. Download or clone this repository
3. Open the project folder in VS Code or any IDE
4. Run the following command:

```bash
python main.py
```
